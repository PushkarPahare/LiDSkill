# Leben in Deutschland -- Kiro Skill

Study companion for the German **"Leben in Deutschland"** citizenship test. Covers all 300 federal questions with narrative study guides and interactive quiz modes.

## What's inside

- **7 study guides** -- one-pagers per topic that cover every testable fact in readable narrative form (~16 min total reading time)
- **Full answer key** -- all 300 answers inline for instant verification
- **Original PDF** -- the complete "Lernkarten Beilage zum Kursbuch" as fallback and quiz source
- **4 modes** -- study, quiz, test (simulated exam), and practice (study + quiz combined)

## Install

Requires [Kiro CLI](https://kiro.dev) -- an AI-powered development assistant that runs in your terminal.

```bash
git clone https://github.com/PushkarPahare/LiDSkill.git ~/.kiro/skills/leben-in-deutschland
```

## Use

In any Kiro CLI session, just say:

| What you say | What happens |
|---|---|
| "Study topic 1" | Reads the Staat & Verfassung guide |
| "Quiz me on Wahlen" | 10 random questions from that category |
| "Practice test" | Simulated exam: 33 questions, pass at 15 |
| "Practice topic 5" | Study NS-Geschichte, then quiz on it |

## About the test

- 33 questions randomly selected from 300
- 60 minutes, multiple choice (4 options each)
- **15 correct** = pass (Orientierungskurstest)
- **17 correct** = counts as Einbürgerungstest

The 300 questions are federal and identical across all Bundesländer. 10 additional state-specific questions (not covered here) may also appear.
